Directrices de marca para todas las piezas

Paleta de colores principal

  • Verde G7Cripto: #0F6E56
  • Negro: #101010 / #1a1a18
  • Blanco: #FFFFFF
  • Verde suave: #E1F5EE (fondos)
  • Acento teal claro: #5DCAA5

Tipografía recomendada

  • Display / Headlines: DM Sans o similar
  • Datos / Mono: DM Mono
  • Alternativa: Inter, Poppins, Plus Jakarta Sans
  • Evitar: tipografías decorativas o poco legibles en mobile

Tono visual general

  • Fintech moderno — no bancario tradicional
  • Oscuro con acentos verdes para TikTok/Meta
  • Limpio y blanco para Google assets
  • UY: estilo más premium y aspiracional
  • ARG: más cripto, más oscuro, más urgente
Reglas de compliance visual: ninguna pieza puede incluir imágenes de contenido adulto o sugestivo. Las piezas son para plataformas de publicidad regulada — el visual debe ser apto para revisión automática de Meta, Google y TikTok. El nicho se reconoce por el copy, no por la imagen.
